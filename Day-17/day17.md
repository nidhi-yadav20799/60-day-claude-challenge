# Day 17 – Vehicle Cost Analysis Dashboard

## Overview

Today I used Claude AI to analyze a vehicle fuel dataset and generate a complete HTML dashboard. The dashboard provides fuel cost analysis, environmental impact, maintenance insights, and E85 fuel economics using business intelligence metrics and interactive SVG visualizations.

---

## Objective

- Analyze vehicle fuel dataset (CSV)
- Calculate fuel efficiency and cost metrics
- Compare different fuel types
- Visualize insights using SVG charts
- Generate a responsive HTML dashboard

---

## Files Included

- `e85_dashboard.html` – Interactive HTML dashboard
- `dashboard_overview.png` – Complete dashboard preview
- `charts_and_kpis.png` – KPI cards and SVG charts
- `fuel_analysis_and_e85_score.png` – Fuel comparison and E85 score

---

## Dashboard Features

- Responsive Glassmorphism UI
- Fuel Cost per Kilometer Analysis
- CO₂ Emission Comparison
- Maintenance Cost Analysis
- Vehicle Age Bucket Analysis
- E85 Fuel Economics
- Break-even Price Calculation
- Monthly Fuel Cost Estimation
- Interactive SVG Charts
- Animated E85 Score Gauge
- Fuel Comparison Cards

---

## Key Insights

- EV has the lowest average running cost per kilometer.
- CNG offers an economical option among combustion fuels.
- E85 produces lower CO₂ emissions but has a slightly higher running cost than Petrol.
- Maintenance costs increase as vehicle age increases.
- Fuel selection depends on running cost, emissions, infrastructure, and driving requirements.
